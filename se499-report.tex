\preface{
\addtocontents{toc}{\vspace{1em}}  % Add a gap in the Contents, for aesthetics

The goal of this work is to add GNU debugging support for \uCPPS and \CFA. To
achieve this goal for \uCPP, new extensions are written to provide additional
support for objects that are unknown to GNU debugger. In addition to the work done for
\uCPP, an addition of a demangler for \CFAS is added to GNU debugger to improve user
experience.

This report assumes readers have some basic knowledge in compiler construction.
Background knowledge about how GNU debugger works and features of \uCPPS and
\CFAS
are provided throughout the report.
}
